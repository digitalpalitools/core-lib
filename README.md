# Digital Pāli Tools - Library

A set of cross platform utilities for Digital Pāli Tools, usable both on frontend and backend.

## Commands

- Build: ```cargo build```
- Test: ```cargo test```
- Format: ```cargo fmt --all -- --check```
- Clippy: ```cargo clippy --tests -- -D warnings```
